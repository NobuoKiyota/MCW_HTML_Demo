# 必要な素材リスト

このディレクトリに以下の画像と音声ファイルを配置してください。

    │   └── click.wav (クリック音)
    ├── japanese/
    │   ├── pond.wav (池の水音 - ループ可能)
    │   ├── shishi.wav (鹿威しの音)
    │   ├── fusuma.wav (ふすまの音)

## 🖼️ 画像ファイルの要件

### 和テーマ画像

1. **room-background.jpg**
   - サイズ: 1400x600px 推奨（またはそれ以上の高解像度）
   - 形式: JPG, PNG
   - 内容: 和室の背景画像（添付画像のような和室と庭園）

2. **pond.png**
   - サイズ: フィールド内の池エリアに収まるサイズ
   - 形式: PNG（透明背景推奨）
   - 内容: 池の水面のオーバーレイ画像

3. **shishi-odoshi.png**
   - サイズ: 80x200px 程度
   - 形式: PNG（透明背景推奨）
   - 内容: 鹿威し（ししおどし）の画像

4. **fusuma.png**
   - サイズ: 120x高さ可変
   - 形式: PNG（透明背景推奨）
   - 内容: ふすま（障子）の画像

5. **bamboo.png**
   - サイズ: 30x200px 程度
   - 形式: PNG（透明背景推奨）
   - 内容: 竹の画像

6. **stone-lantern.png**
   - サイズ: 60x100px 程度
   - 形式: PNG（透明背景推奨）
   - 内容: 石灯籠の画像

### サイバーテーマ画像

- 現在はCSSで描画していますが、必要に応じて画像を追加可能

## 🔊 音声ファイルの要件

### 共通音声

1. **hover.wav**
   - 長さ: 0.1-0.2秒
   - 内容: ホバー時の短い音

2. **click.wav**
   - 長さ: 0.1-0.3秒
   - 内容: クリック時の短い音

### 和テーマ音声

1. **pond.wav**
   - 長さ: 0.5-2秒（ループ可能）
   - 内容: 水が「ちゃぱちゃぱ」と鳴る音

2. **shishi.wav**
   - 長さ: 0.5-1秒
   - 内容: 鹿威しが「カーン」と鳴る音

3. **fusuma.wav**
   - 長さ: 0.2-0.5秒
   - 内容: ふすまを動かす「ガラッ」という音

4. **Japanese01.mp3**
   - 長さ: 1分以上（ループ再生）
   - 形式: **MP3** (推奨)
   - 内容: 和風のBGM

5. **Japanese02.mp3**
   - 長さ: 1分以上（ループ再生）
   - 形式: **MP3** (推奨)
   - 内容: 2曲目のBGM

6. **amb.wav**
   - 長さ: 10-30秒（ループ再生）
   - 内容: 環境音（水の流れる音、風の音など）

### サイバーテーマ音声

1. **hologram.wav**
   - 長さ: 0.3-1秒
   - 内容: ホログラムエフェクトの電子音

2. **bgm-01.wav**
   - 長さ: 1分以上（ループ再生）
   - 内容: サイバーなBGM（電子音楽、ビート）

3. **bgm-02.wav** (オプション)
   - 長さ: 1分以上（ループ再生）
   - 内容: 2曲目のBGM

4. **amb.wav**
   - 長さ: 10-30秒（ループ再生）
   - 内容: 環境音（電子ノイズ、機械音など）

## 📝 注意事項

- **音声形式**:
    - 効果音（SE）や環境音は **WAV形式** を推奨します。
    - BGMのような長時間の音声はファイルサイズ削減のため **MP3形式** が推奨されます（ブラウザでの再生もスムーズです）。
- ループ再生する音声は、ループ時に途切れないように編集してください。
- 画像ファイルは、ファイルサイズを考慮して適切な形式（JPG/PNG）を選択してください。
- 透明背景が必要な画像はPNG形式を使用してください。

## 🎨 素材の入手方法

- **無料素材サイト**: Pixabay, Pexels, Unsplash, Freesound.org など
- **AI生成**: Midjourney, DALL-E, Stable Diffusion, Suno/Udio (音楽) など
- **自作**: 写真撮影、イラスト作成、音声録音など
